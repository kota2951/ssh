'''text
[# 目的
社会学・数理経済学におけるエージェントベースモデリング（ABM）の手法を用いて、「学歴構造」「偶発的な運（幸運・不運）」「実力（初期資質と経年努力）」の3つの社会因子が、40年間の社会人人生を通じて個人の最終的な資産格差（不平等）にどのように影響を与えるかを数理的に検証するプログラムを作成してください。

# 基礎モデルの指定
2022年イグノーベル経済学賞を受賞した Alessandro Pluchino 教授らの『Talent vs Luck（TVLモデル）』のロジック（「才能は正規分布するが、富の分布はパレート分布になる」という性質）をベースとし、これを現代日本社会の階層構造を模した高度モデルへと拡張します。

# 基本ルール・条件設定
1. エージェント総数（N）：10,000名
2. シミュレーション期間（T）：40年間（社会人人生 22歳〜62歳を想定）
3. フェーズの分離：
   - 「静的初期化フェーズ（誕生〜22歳）」：初期条件（学歴クラス、初期才能、初期資産）をマクロ統計に基づいて固定割り当て。
   - 「動的時間発展フェーズ（22歳〜62歳）」：40年間を1年ステップでループ計算し、動的変数を更新。

# 変数と社会パラメータの設定
1. 初期才能（initial_talent）：平均0.6、標準偏差0.1の正規分布で生成（[0.0, 1.0]にクリップ）。
2. 学歴構造（education）：以下の日本のリアルな統計比率に準拠し、エージェントを6グループに排他的に割り当てる。高校卒業（HighSchool）の値を「1.0」の基準とし、各パラメータを相対値としてマッピングする。
   - 義務教育卒 (Compulsory) : 比率0.8% / 初期資産0.6 / 幸運時1.02倍 / 不運時残存率90% / 実力成長率 -0.002
   - 高校卒業 (HighSchool) : 比率14.5% / 初期資産1.0 / 幸運時1.03倍 / 不運時残存率92% / 実力成長率 +0.000
   - 専門学校卒 (Vocational) : 比率13.0% / 初期資産1.2 / 幸運時1.04倍 / 不運時残存率93% / 実力成長率 +0.001
   - 短大高専卒 (JuniorCollege) : 比率13.1% / 初期資産1.2 / 幸運時1.04倍 / 不運時残存率93% / 実力成長率 +0.001
   - 大学卒業 (University) : 比率51.2% / 初期資産2.0 / 幸運時1.05倍 / 不運時残存率95% / 実力成長率 +0.003
   - 大学院修了 (GradSchool) : 比率7.4% / 初期資産3.0 / 幸運時1.06倍 / 不運時残存率96% / 実力成長率 +0.005

# 毎年の時間発展ループ内の高度アルゴリズム
1. 実力の自己投資：毎年、実力に「実力成長率」を加算（[0.0, 1.0]にクリップ）。高学歴ほど伸び、義務教育卒は経年陳腐化する。
2. 累積的優位（動的マタイ効果）：幸運遭遇確率（P_luck）は、ベースの0.15に対し、個人の資産が社会平均を上回る割合に応じて最大+0.05まで動的にブーストされる。
   数式： P_luck = 0.15 + clip((個人資産 / 社会平均資産) * 0.01, 0.0, 0.05)
3. 幸運の判定：確率 P_luck で幸運に遭遇し、かつその瞬間に引いた乱数が自分の現在の実力以下であった場合のみ成功。資産が「幸運時リターン倍率」で複利増加。
4. 不運の判定と構造的排除：一律 0.15 の確率で不運に遭遇。資産は「不運時残存率」に減少。さらに階層転落トラップとして、義務教育卒および高卒が不運に遭遇した場合、翌年以降の実力（talent）が直接「-0.02」強制ペナルティとして削り取られる。

# 出力および統計分析の仕様
1. 文字による詳細レポート出力：
   - 決定係数（R²）の表示。
   - 最終資産を対数変換（log10）して目的変数とし、初期才能、最終才能、幸運成功数、不運遭遇数、および高卒を基準（=0）とした他の5つの学歴ダミー変数を説明変数とする多重共線性を考慮した「標準化偏回帰分析（線形回帰）」を実行。影響度（Beta値）を降順でランキング表示。
   - 学歴別の最終資産平均値を表示。
   - 最終資産額の「トップ3保有者」の個別プロファイル（学歴、初期才能、最終実力、幸運成功回数、不運遭遇回数、最終資産額）をケーススタディ用に出力。
2. 可視化（2つのグラフを左右に並べて表示）：
   - グラフ①（資産順ソート能力散布図）：全10,000名のエージェントを最終資産の昇順（貧乏な順から金持ちの順）に並べ替える。横軸をその資産順位（Agent Rank：0〜9999）、縦軸を最終能力値（0.0〜1.0）、プロットの色を学歴（6色）とした、黒い折れ線のない純粋な「能力値分布の散布図」を描画。
   - グラフ②（学歴別平均資産棒グラフ）：各学歴の最終資産平均値をシンプルな棒グラフで可視化し、各棒の上に具体的な数値を太字で表示。

これらすべての処理を、Pythonの pandas, numpy, matplotlib, seaborn, sklearn を使って美しく実装してください。]
